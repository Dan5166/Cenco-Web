¿Que nubes se utilizan en Cencosud?

1. {{cloud_1}}

Microsoft Azure es una plataforma de servicios en la nube ofrecida por Microsoft. Esta plataforma ofrece una amplia variedad de servicios de infraestructura, plataforma y software como servicio (IaaS, PaaS y SaaS, respectivamente). Azure también es compatible con una amplia variedad de lenguajes de programación, lo que la hace una opción atractiva para empresas que trabajan con múltiples tecnologías.

2. {{cloud_2}}

AWS es una plataforma de servicios en la nube ofrecida por Amazon. Esta plataforma ofrece una amplia variedad de opciones para almacenamiento, procesamiento de datos y desarrollo de aplicaciones.

3. {{cloud_4}}

Oracle Cloud es una plataforma de servicios en la nube ofrecida por Oracle Corporation. Esta plataforma proporciona una amplia variedad de servicios de infraestructura, plataforma y software como servicio (IaaS, PaaS y SaaS, respectivamente). Oracle Cloud puede resultar conveniente para empresas que buscan soluciones de nube personalizadas y escalables para sus necesidades de tecnología empresarial. Además, Oracle Cloud ofrece soluciones específicas para diferentes sectores, como finanzas, salud y comercio minorista.

4. {{cloud_3}}

Google Cloud Platform es una plataforma de servicios en la nube ofrecida por Google. Esta plataforma ofrece una amplia variedad de opciones para almacenamiento, procesamiento de datos y desarrollo de aplicaciones. Google también ha desarrollado soluciones específicas para diferentes sectores, como el análisis de datos y el aprendizaje automático.
